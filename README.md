# MTN-Mobile-USSD
This is a look alike of the MTN dial
when a user runs the code, the program requests them to input there desired option, whether to send money, withdraw or make payments, then they continue with the procedures until they are satisified with their demands, and on finishing, they opt out by dialing 0 on the main menu. 
. This session-based interaction allows users to navigate menus or perform transactions like in real time.
